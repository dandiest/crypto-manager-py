## Features
- **Add Assets**: Save your coin purchases with quantity and buy price.
- **Auto-Calculation**: Automatically calculates the total value of each asset and the entire portfolio.
- **Persistence**: Data is saved locally in a SQLite database.
- **Clean UI**: Formatted tables for easy reading.

## How to use
1. Clone the repository: `git clone https://github.com/dandiest/CryptoPortfolioManager.git`
2. Run the script: `python crypto_manager.py`
3. Follow the on-screen menu.

## Tech Stack
- Python 3.x
- SQLite3
